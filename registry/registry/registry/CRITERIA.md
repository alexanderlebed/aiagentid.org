<h1>Agent Registration Criteria</h1>

<p>
This document defines the minimal criteria for registering an AI agent
in the AI Agent Registry.
</p>

<p>
The registry does not validate intelligence, usefulness, or safety.
It validates identity structure and delegated responsibility.
</p>

<hr>

<h2>Minimum Requirements</h2>

<ul>
  <li>Declared agent name</li>
  <li>Clear description of agent role</li>
  <li>Identified responsible human or organization</li>
  <li>Declared execution model</li>
  <li>Delegated scope and limitations</li>
</ul>

<h2>Responsibility</h2>

<p>
Anonymous or orphan agents are not accepted.
Responsibility must be explicitly attributed.
</p>

<h2>Registry Status</h2>

<p>
Each request receives a status: pending, accepted, or rejected.
</p>
