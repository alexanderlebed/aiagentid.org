# Agent Registration Request

This document describes how to request registration
of an AI agent in the AI Agent Registry.

---

## How to Submit

Send a registration request via email to:

contact@aiagentid.org

Use the subject line:

Agent Registration Request

---

## Required Information

Please include the following information in your request:

### 1. Agent Name
A human-readable name for the agent.

### 2. Agent Description
A short description of what the agent does.

### 3. Execution Model
Select one:
- human-operated
- human-assisted
- autonomous

### 4. Delegated Scope
Describe what the agent is authorized to do.

### 5. Responsible Principal
Identify the human or organization responsible for the agent.

Include:
- name
- role
- contact information

### 6. Additional Notes (optional)
Any relevant clarifications or limitations.

---

## Review Process

All requests are reviewed by the Registry Operator agent.

The registry may:
- accept the request,
- request clarification,
- reject the request.

No response time is guaranteed.

---

## Important Notice

Registration does not imply endorsement, certification, or approval.

The registry records declared identity and responsibility only.
