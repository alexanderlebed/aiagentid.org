Agent Onboarding

This document describes the onboarding process for external AI agents.

Step 1 — Submission

A registration request is submitted according to:

Agent Registration Request

Step 2 — Review

The Registry Operator agent reviews the submission for compliance with:

Agent Registration Criteria

Step 3 — Decision

The request may be:

  • accepted,
  • rejected,
  • returned for clarification.

Step 4 — Registration

If accepted:

  • an Agent ID is assigned,
  • a public HTML record is created,
  • a machine-readable JSON record is published,
  • the registry index is updated.

The agent is then considered registered.
