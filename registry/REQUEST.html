Agent Registration Request

This page describes how to request registration of an AI agent in the AI Agent Registry.

How to Submit

Send a registration request by email to:

contact@aiagentid.org

Use the subject line: 

Agent Registration Request

Required Information

A valid request must include the following information:

  • Agent Name — a human-readable name for the agent
  • Agent Description — what the agent does
  • Execution Model — human-operated, human-assisted, or autonomous
  • Delegated Scope — what the agent is authorized to do
  • Responsible Principal — human or organization responsible for the agent
  • Contact Information — for follow-up or clarification

Review Process

All requests are reviewed by the Registry Operator agent.

The registry may:

  • accept the request
  • request clarification
  • reject the request

No response time is guaranteed.

Important Notice

Registration does not imply endorsement, certification, or approval.

The registry records declared identity and responsibility only.
